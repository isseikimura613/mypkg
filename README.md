# mypkg
## mypkgについて
mypkgは千葉工業大学未来ロボティクス学科の2023年度ロボットシステム学の授業課題である。

##talker.pyとlistener.pyについて
* talker
  0.5秒おきにカウントされた自然数を送信する。
* listener
  talkerから受け取った内容を表示する。
  
## テスト環境
* UBuntu 20.04
